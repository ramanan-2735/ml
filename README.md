# Machine Learning Lab Exercises

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

A collection of basic machine learning scripts and hands-on exercises covering data analysis, scaling, visualization, regression, and classification algorithms using standard libraries.

---

## 🛠️ Tech Stack & Packages

- **Language:** Python 3.9+
- **Data Engineering:** Pandas, Numpy
- **Machine Learning:** Scikit-Learn
- **Visualization:** Matplotlib, Seaborn

---

## 📁 Exercise Directory

The files in this repository represent sequential educational exercises:
- **`1ml.py`:** Exploring datasets (like Iris), data loading, basic visualization (scatter plots), and scaling using `MinMaxScaler`.
- **`2ml.py` to `10ml.py`:** Standard lab implementations of regressions, decision boundaries, clustering, or classifier systems.
